# Working linux distributions

At the moment kanku works only for 

* openSUSE Tumbleweed

* openSUSE Leap 42.2

* openSUSE Leap 42.1

But kanku is pure perl + libvirt + qemu, so it should run on any linux distribution.
but feel free to send us your patches
