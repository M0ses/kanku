NAME

Kanku::Notifier::Sendmail

SYNOPSIS

  notifiers:
    - 
      use_module: Kanku::Notifier::Sendmail
      options:
        from: kanku@suse.de
        to: kanku-user@opensuse.org
      states: failed,succeed

DESCRIPTION

This notifier module allows you to send an email to a given list of recipients
