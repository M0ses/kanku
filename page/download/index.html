Download

Sources

To get the latest development code you can clone https://github.com/M0ses/kanku

git clone https://github.com/M0ses/kanku

Releases

You can download the sources of the latest release as tar/zip from our releases page

https://github.com/M0ses/kanku/releases

Packages

We also provide packages for openSUSE/Fedora/Ubuntu/Debian.

  • Released Packages - https://build.opensuse.org/project/show/devel:kanku
  • Testing Packages - https://build.opensuse.org/project/show/devel:kanku:staging

openSUSE Tumbleweed users can also use zypper to install.
